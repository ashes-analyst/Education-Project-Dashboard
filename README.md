# Education-Project-Dashboard
# 📊 Power BI Dashboard: Foundational Learning Intervention Project

## 🔍 Project Overview
This Power BI dashboard presents insights from an education intervention focused on **out-of-school, irregular, and learning-deficient children from marginalized communities** across five locations in Bihar: **Chenari, Sasaram, Piro, Raxaul, and Gahiri**.  

The project addresses critical gaps in **Foundational Literacy and Numeracy (FLN)** and aims to **mainstream children into formal education systems** through targeted, community-based learning support.

---

## 🎯 Objective of the Dashboard
The purpose of this dashboard is to:
- Track learning progress of children across key FLN indicators  
- Monitor children’s progress over time  
- Identify learning gaps for targeted interventions  
- Support data-driven decision-making for program improvement  

---

## 📚 About the Program
The intervention operates through **Flexible Learning Centres (FLCs)** conducted in **morning and evening sessions**, ensuring accessibility for children with diverse constraints.

### Key Features:
- 🎓 Focus on **dropout and irregular children**
- ✏️ Emphasis on **age-appropriate learning levels**
- 📖 Strengthening **basic reading and numeracy skills**
- 🎲 Use of **play-based learning approaches**
- 🧩 Integration of **Teaching Learning Materials (TLMs)** and rhymes
- 👩‍🏫 **Trained tutors** providing need-based, personalized support  

---

## 📊 Dashboard Highlights
This dashboard provides:

### 📌 Enrollment Overview
- Total number of children enrolled  

### 👥 Demographic Insights
- Gender-wise distribution of children  
- Location-wise analysis across **five project areas (Chenari, Sasaram, Piro, Raxaul, Gahiri)**  

### 📈 Learning Level Classification
- Categorization of children into **Group A and Group B** based on learning levels  

### 📉 Learning Progress & Outcomes
- Improvement trends in learning levels over time  
- Subject-wise progress in **Literacy and Numeracy**

- Significant improvement observed in **basic reading ability** after intervention  
- Higher retention in centres using **activity-based learning methods**

- Targeted support helps accelerate progress for **low-performing learners**
